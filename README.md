# ECHO Raid Pack

[Download latest release](https://github.com/Meriksen95/ECHO-Raid-Pack/releases/latest/download/ECHO-Raid-Pack.taco)
Please note that this pack is optimized for BlishHUD. TaimiHUD and Burrito may not be able to toggle markers without using the menu.

This pack is made for Guild Wars 2, specifically for the ECHO Chamber guild and its raid group.

Feel free to join our [Discord](https://discord.gg/mzHMCqSrcu)

These are the markers included in this pack, in addition to the well-known in-game symbols.

### From left to right:
1. Information marker (stand close for information and/or to toggle markers)
2. Location marker (stand here)
3. Target marker (aim/jump/land here)
4. Leap marker
5. Bugging marker (used to bug Glenna around)
6. Stealth marker
7. Portal enter
8. Portal exit
